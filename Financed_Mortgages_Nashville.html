Method. Every point is a Davidson County parcel pledged as collateral on a recorded Deed of Trust (mortgage) taken by an institutional single-family operator. Color = parent operator; dot size scales with the loan amount on that securitization. Loan amounts are the recorded consideration on the deed of trust; one securitized loan covers many homes, so the same amount attaches to every parcel in its pool. Borrowers resolved to parent via the operator name crosswalk (e.g. MILE HIGH / OLYMPUS / TRUE NORTH → Progress; STAR → Starwood; SWH / CSH → Colony-Starwood/Invitation; SFR JV / TENAX → Tricon). Source: Davidson County Register of Deeds (instrument type DOT) + ParcelHistory ownership (Layer 2) + Cadastral coordinates.
