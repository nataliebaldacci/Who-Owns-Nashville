Methodology. Each dot is a residential rental parcel placed in the year it was first registered with Metro Codes & Building Safety (ePermits landlord registration, caseTypeID 176), 2008–2026. Corporate-owned = current owner passes the entity-keyword test (LLC, Inc, LP, Trust…). Single-family = land use SINGLE FAMILY / CONDO / ZERO LOT LINE / DUPLEX with 1–2 dwelling units (Davidson cadastral). Sources: Metro ePermits + ParcelHistory ownership (Layer 2) + Cadastral (Layer 4). Mandate: Metro Code § 16.33.170 & Tenn. Code Ann. § 66-28-107. Full citations: data sources. Davidson County.
