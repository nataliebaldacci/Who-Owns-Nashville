# Who Owns Nashville?

Interactive maps of institutional and corporate ownership of residential housing in
Davidson County, Tennessee, built entirely from primary Metro Nashville government data
(ePermits, ParcelHistory, Cadastral). By Natalie Baldacci, Vanderbilt Law School.

**Live site:** https://nataliebaldacci.github.io/Who-Owns-Nashville/

## Maps
- `WhoOwnsNashville_Explorer.html` — search any owner's full portfolio
- `Corporate_Landlords_Timeline_LandlordReg.html` — corporate single-family rental growth 2008–2026 (cumulative vs. active)
- `Corporate_Landlords_Timeline_Nashville.html` — corporate acquisition timeline
- `Corporate_Landlords_ParcelShapes_Nashville.html` — corporate-owned parcels as lot shapes, by operator
- `Corporate_Landlords_Hexmap_ckmeans.html` — concentration hexmap
- `BTR_Current_vs_Pipeline_Map.html` — build-to-rent: built vs. in the pipeline
