Holdings = parcels whose current cadastral owner resolves to an operator through the project's canonical parent-brand crosswalk, which maps shell entities to their parent (e.g. SFR JV-1/JV-2 → Tricon, YAMASA / PR Borrower / FYR → Progress, Main Street Renewal → Amherst). Registration = a currently-active landlord registration on the parcel (ePermits caseTypeID 176; Metro Code § 16.33.170, T.C.A. § 66-28-107). Source: Cadastral (L4) + canonical ownership attribution + ePermits. Davidson County.
